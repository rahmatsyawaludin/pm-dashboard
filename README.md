## Project Overview

This dashboard extends my original project, **Community-Based Offline Learning Companion App**, a digital education initiative designed to support learning access in communities with limited internet connectivity.

The original project focused on strategic planning, stakeholder mapping, implementation design, and project scope definition. To move beyond concept planning, I developed this **Project Management Executive Dashboard** to simulate real-world project execution monitoring.

Using Python, Streamlit, and Google Sheets, I created a live PM reporting system that tracks project performance across schedule, budget, risks, workload, and delivery phases—similar to how project managers report to leadership and stakeholders.

This project demonstrates how strategic planning can be translated into operational control and executive decision-making.

---

## Objective

The goal of this dashboard is to provide a centralized project monitoring system for:

- Schedule tracking and delivery status
- Budget allocation and spending variance
- Risk identification and severity analysis
- Team workload visibility
- Phase completion monitoring
- Executive reporting and stakeholder communication

This reflects practical project management processes used in real implementation environments.

---

## Key Features

### KPI Executive Cards

Quick visibility of:

- Total budget
- Spent to date
- Budget variance
- Overall project completion
- Active risks
- Open work items

### Gantt Chart Timeline

Visual project scheduling with:

- Start and end dates
- Task dependencies
- Delivery timeline monitoring
- Milestone visibility

### Budget Variance Analysis

Tracks:

- Planned vs actual spending
- Cost overrun detection
- Financial risk indicators

### Risk Heat Map

Monitors project risks by:

- Severity
- Probability
- Business impact
- Priority for mitigation

### Phase Completion Dashboard

Shows progress across:

- Initiation
- Planning
- Execution
- Monitoring
- Closure

### Team Workload Tracking

Provides visibility into:

- Resource allocation
- Task ownership
- Capacity balancing
- Team performance monitoring
